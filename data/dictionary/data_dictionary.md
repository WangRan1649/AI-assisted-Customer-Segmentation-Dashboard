# Data Dictionary

## customer_segments.csv

| Field | Meaning |
|---|---|
| segment | Customer segment name |
| share | Percentage of total customers |
| share_pct | Numeric segment share percentage |
| customer_count | Number of customers in the segment |
| business_role | Business role of the segment |
| frequency_range | Purchase frequency range |
| monetary_range | Spending or AOV range |
| weighted_aov | Segment-level weighted average order value |
| avg_recency_days | Average days since last login |
| avg_rfm_score | Average RFM score for the segment |
| avg_value_proxy_score | Average 0-100 value proxy score |
| avg_recency_score | Average recency score, where higher is better |
| avg_frequency_score | Average frequency score, where higher is better |
| avg_monetary_score | Average monetary score, where higher is better |
| category_preference | Preferred product categories |
| key_issue | Main business issue |
| action_priority | Strategic action direction |

## powerbi_llm_insights.csv

| Field | Meaning |
|---|---|
| insight_title | Short title for the AI-assisted insight |
| insight_text | Business-readable insight text |
| segment_name | Related customer segment or cross-dimensional scope |
| priority | Review priority for business users |
| review_status | Human review status |
| generated_at | Pipeline run timestamp in UTC |

## run_metadata.json

| Field | Meaning |
|---|---|
| run_time_utc | Pipeline run timestamp in UTC |
| raw_files | Raw CSV files read by the pipeline |
| raw_row_count | Total raw rows read before cleaning |
| cleaned_row_count | Rows retained after cleaning |
| processed_customer_count | Unique customers processed |
| requested_provider | Provider requested through `.env` or CLI |
| provider | Actual provider used after fallback logic |
| model | LLM or mock model identifier |
| api_reached | Whether the SiliconFlow API returned a response |
| validation_passed | Whether the LLM numeric guardrail passed |
| fallback_used | Whether SiliconFlow failed and Mock was used |
| error | Provider error message, if any |
| output_files | Files generated by the pipeline |
